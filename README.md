# underpressure

Blood pressure tracker for Android

## Features

- Track blood pressure with up to 4 time slots per day
- Record measurements in `sys/dia@pulse` format
- Clean, distraction-free table view
- Pressure line chart with sharing option
- Share measurements via text message or CSV export
- All data stays on your device - no sync, no servers, no cloud
- No ads, no payments, no in-app purchases, no trials
- Completely free and open source (MIT License)
